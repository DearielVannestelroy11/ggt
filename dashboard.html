GlowScan AI merupakan aplikasi berbasis kecerdasan buatan yang dirancang untuk melakukan pemindaian visual secara real-time menggunakan kamera perangkat. Sistem ini memanfaatkan teknologi pengolahan citra digital untuk menghasilkan efek visual glow serta analisis wajah secara otomatis.

Aplikasi ini dikembangkan menggunakan framework Flask sebagai backend, OpenCV sebagai pustaka pengolahan citra, serta antarmuka web modern yang dilengkapi dengan autentikasi pengguna dan dashboard interaktif.

GlowScan AI diharapkan menjadi media pembelajaran dan pengembangan awal dalam bidang kecerdasan buatan, khususnya pada implementasi computer vision berbasis web.
